Minu bitcoin kalkulaator

Minu bitcoin kalkulaator võtab kasutaja sisestatud bitcoini koguse ja arvutab selle hinna kasutaja sisestatud valuutas(EUR, USD, GBP, EEK)

Kalkulaatori tööprotsess:

Kalkulaator konverteerib kasutaja sisestatud BTC koguse valitud valuutasse. Valikus on neli valuutat: EUR, USD, GBP ja EEK.
Esiteks kontrollib programm kas kõik vajalikud väljad (bitcoini kogus ja sihtvaluuta) on täidetud. Probleemi korral
väljastab veateate, et mõni sisend on puudu. Kui kõik sisendid on olemas, siis kalkulaator esitab päringu bitcoin API
lehele COINDESK valuuta väärtuse kohta valitud valuutas välja arvatud EEK.
EEK puhul võetakse API'ist EUR'i väärtus ja korrutatakse 15,6466'ga. Programm kalkuleerib kokku API andmed valuuta osas
ning esitab need akna all vasakus nurgas.
